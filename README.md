# 📊 Data Analysis Projects (Python | Jupyter Notebook)

This repository contains multiple **data analysis projects using Python**.
The projects focus on **Exploratory Data Analysis (EDA), data visualization, and business insights generation** using real-world datasets.

The analysis is performed using **Python, Pandas, NumPy, Matplotlib, and Seaborn in Jupyter Notebook**.

---

# 📁 Projects Included

## 1️⃣ Sales Data Analysis Project

### 📌 Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** of a sales dataset to understand **customer purchasing behavior and product trends**.

The goal is to analyze demographic factors such as **gender, age group, marital status, occupation, location, and product categories** to generate meaningful business insights.

---

### 🔗 Project File

Notebook:

https://github.com/Faisalshahzad03/data-analysis-with-python/blob/main/Sales%20%20project%20.ipynb

---

### 🎯 Objectives

* Clean and preprocess sales data
* Perform exploratory data analysis (EDA)
* Identify key customer segments
* Analyze sales trends and purchasing behavior
* Generate actionable business insights

---

### 🛠️ Tools & Technologies

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

### 📊 Analysis Performed

* Gender-based purchase analysis
* Age group analysis
* State-wise sales distribution
* Marital status insights
* Occupation-based purchasing trends
* Product category performance
* Top selling products analysis

---

### 📈 Key Insights

* Female customers showed higher purchasing activity
* Age group **26–35** contributed significantly to sales
* Major sales came from specific regions
* IT, Healthcare, and Aviation professionals were major buyers
* Food, Clothing, and Electronics were top-selling categories

---

## 2️⃣ Hospitality Data Analysis Project

### 📌 Project Overview

This project analyzes hospitality data for **Royal Thames Hotel Group** across different UK cities.

The goal is to evaluate **hotel performance metrics such as occupancy rate, revenue trends, booking platforms, and room category performance**.

This analysis helps identify patterns that can improve **hotel pricing strategies and operational decisions**.

---

### 🔗 Project File

Notebook:
https://github.com/Faisalshahzad03/data-analysis-with-python/blob/main/Hospitality%20project.ipynb

---

### 🎯 Objectives

* Clean and preprocess hospitality data
* Perform exploratory data analysis (EDA)
* Analyze hotel performance metrics
* Identify booking platform trends
* Understand room category performance
* Generate insights to improve hotel revenue and occupancy

---

### 🛠️ Tools & Technologies

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

### 📊 Analysis Performed

* Data cleaning and preprocessing
* Booking platform distribution analysis
* Room category exploration
* Hotel performance metrics analysis
* Occupancy rate trends
* Revenue and ADR analysis
* City-wise hotel performance
* Customer booking behavior insights

---

### 📈 Key Insights

* Certain booking platforms generated the majority of reservations
* Premium room categories produced higher revenue per booking
* Occupancy varied across cities and hotel properties
* Seasonal trends influenced booking volume and revenue
* Optimizing pricing strategies could improve ADR and occupancy
